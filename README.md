<h2>Запуск и сборка приложения </h2>

<p>Для установки проекта</p>

```bash
yarn add
```

<p>Для запуска</p>

```bash
yarn dev
```
<p>Для сборки</p>

```bash
yarn build
```

<h2>Запуск тестов</h2>

```bash
yarn test
```
